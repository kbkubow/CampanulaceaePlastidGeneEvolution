This is a description of the dNdSdatasheet.csv

Most of the data in this datasheet comes from extracting data from PAML output using the treeio package in R.

dN_vs_dS_B is the corrected version of dN_vs_dS where nodes that had either a dN or dS of 0 had dN_vs_dSB set as missing (uncalculable).
For all other nodes, dN_vs_dS_B is the same as dN_vs_dS.
